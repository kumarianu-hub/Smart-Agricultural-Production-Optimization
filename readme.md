# OptiCrop – Smart Agricultural Production Optimization

## Overview
OptiCrop is a Flask-based web application that recommends suitable crops based on soil type and season. It also predicts crop yield based on the cultivated land area.

## Features
- Crop Recommendation
- Yield Prediction
- Farmer Data Storage
- SQLite Database
- Simple Web Interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

## Installation

Install the dependencies:

pip install -r requirements.txt

Run the project:

python app.py

Open your browser:

http://127.0.0.1:5000/
